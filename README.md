# DataLens Renderable

Modular DataLens notebook that scans `data`, recursively extracts archives, analyses supported file types, and renders representative files.

Renderable types:

- `.shp`, `.gpkg`, `.geojson`
- `.tif`, `.tiff`
- `.png`, `.jpg`, `.jpeg`, `.webp`
- `.pdf`
- `.csv`, `.tsv`, `.xlsx`, `.parquet`
- `.txt`, `.md`, `.json`
